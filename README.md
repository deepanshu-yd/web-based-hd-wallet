# Web-Based Hierarchical Deterministic (HD) Wallet

A web-based HD wallet that supports both **Solana** and **Ethereum** blockchain. Generate a single seed phrase and derive multiple wallet addresses for both chains.

## npm libraries used:

- **bip39** - Mnemonic generation
- **ed25519-hd-key** - Solana key derivation
- **@solana/web3.js** - Solana wallet utilities
- **ethers** - Ethereum wallet utilities
- **tweetnacl** - Cryptographics function

**Warning**: This is a practice project, never used real funds. UI development and more features in progress

## Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/web-based-hd-wallet.git

# Navigate to project directory
cd web-based-hd-wallet

# Install dependencies
npm install

# Start development server
npm run dev
```
